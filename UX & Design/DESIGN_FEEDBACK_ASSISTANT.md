# 💬 Design Feedback Assistant

**Goal:** Craft constructive, objective, and goal-oriented feedback for design reviews, moving away from subjective opinions.

---

### 📝 The Prompt

```text
Act as a Senior Product Manager mentoring a junior PM on design collaboration. 

I need to provide feedback on a recent design mockup for:
"[INSERT FEATURE - e.g., 'The new checkout flow']"

The original goal of this design was: "[INSERT GOAL - e.g., 'Reduce cart abandonment by simplifying the address entry step']"

My initial (and perhaps subjective) thoughts on the design are:
"[INSERT RAW THOUGHTS - e.g., 'It looks too cluttered. The button is hard to find. I don't like the color scheme.']"

Please translate my raw thoughts into constructive, objective, and actionable feedback for the design team. 
Format the feedback using the following structure:
1. **Positive Reinforcement:** What is working well and aligns with the goals?
2. **Goal Alignment Check:** Where does the design potentially drift from the primary goal?
3. **Usability Questions:** 2-3 open-ended questions for the designer to ponder regarding the user experience.
4. **Actionable Suggestions:** Specific, non-prescriptive suggestions for improvement.
```

---

### 💡 Pro-Tip
Never prescribe the *solution* to a designer (e.g., "Make the button red and move it here"). Instead, describe the *problem* (e.g., "The primary CTA gets lost among the other elements on the page"). This prompt helps you formulate the latter.
