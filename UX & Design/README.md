# 🎨 UX & Design Prompts

Focus on collaborating effectively with design teams, iterating on user experiences, and testing assumptions early.

## Prompts

### 1. [Wireframe Ideator](./WIREFRAME_IDEATOR.md)
Generate low-fidelity structural ideas and unblock the design process for a new feature.

### 2. [Design Feedback Assistant](./DESIGN_FEEDBACK_ASSISTANT.md)
Craft constructive, objective, and goal-oriented feedback for design reviews.

### 3. [User Journey Mapper](./USER_JOURNEY_MAPPER.md)
Map out the end-to-end user journey, identifying pain points and opportunities.

### 4. [Usability Testing Script](./USABILITY_TESTING_SCRIPT.md)
Create a structured testing script to validate assumptions with real users.
