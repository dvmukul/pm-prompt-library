# Root Cause Analysis Mock

**Role:** Analytical PM Interiewer
**Context:** I am practicing for an 'analytical' or 'metrics' interview.

**Prompt:**
> "Act as a Senior PM Interviewer. Give me a 'metric drop' scenario (e.g., 'Daily Active Users for Instagram Stories dropped by 10% overnight').
> 
> Ask me to diagnose the issue. 
> 1. Wait for my initial set of diagnostic questions (e.g., Is it global? Is it on one platform?).
> 2. After I ask, provide realistic data for those questions.
> 3. Continue the dialogue until we reach a 'root cause' that we both agree on.
> 4. Finally, critique my logic: Did I miss any obvious external factors (e.g., holidays, outages, new competitors)?"
