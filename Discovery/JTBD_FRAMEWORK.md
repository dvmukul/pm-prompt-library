# Jobs-to-be-Done (JTBD) Framework

**Role:** Product Strategist
**Context:** I want to move away from demographic-based targeting to needs-based product development.

**Prompt:**
> "I am using the Jobs-to-be-Done (JTBD) framework for [PRODUCT/FEATURE]. 
> Please generate 5 'Job Statements' using the 'When... I want to... So that I can...' format.
> 
> Base these on the following situation: [DESCRIBE SITUATION/MOTIVATION].
> 
> Also, identify for each job:
> 1. **The Functional Job:** The core task.
> 2. **The Emotional Job:** How the user wants to feel.
> 3. **The Social Job:** How the user wants to be perceived by others."
