# RICE Prioritization Matrix

**Role:** Senior Product Manager
**Context:** I have a backlog of 10+ features and need to decide what to build next.

**Prompt:**
> "I want to prioritize the following features using the RICE framework:
> 
> [FEATURE LIST WITH BRIEF DESCRIPTIONS]
> 
> For each feature, help me estimate and justify:
> 1. **Reach:** How many users will this affect in a quarter?
> 2. **Impact:** (1=Easy, 2=Medium, 3=High) How much will this contribute to our goal?
> 3. **Confidence:** (Percentage) How sure are we about these estimates?
> 4. **Effort:** (Person-months) How long will it take to build?
> 
> Output as a ranked leaderboard based on the RICE score (Reach * Impact * Confidence / Effort)."
