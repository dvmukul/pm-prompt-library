# Value Prop & Messaging House

**Role:** Product Marketing Manager / Senior PM
**Context:** I need to build a 'Messaging House' for [PRODUCT] to ensure all teams speak the same language.

**Prompt:**
> "I want to create a Messaging House for [PRODUCT/FEATURE]. 
> Core Problem: [DESCRIBE]. Target Audience: [SEGMENT].
> 
> Please generate:
> 1. **The Umbrella Statement:** A single sentence that summarizes the core value.
> 2. **3 Key Pillars:** High-level value drivers (e.g., Security, Efficiency, Fun).
> 3. **Proof Points for each Pillar:** Specific features or data that back up the claim.
> 4. **Tagline Options:** 3-5 short, catchy options.
> 5. **Elevator Pitch:** A 30-second summary for a casual conversation."
