# Cohort Analysis Guide

**Role:** Growth Product Manager
**Context:** Our retention numbers look steady, but I want to see if recent changes improved new user stickiness.

**Prompt:**
> "Help me structure a cohort analysis for [PRODUCT].
> 
> 1. **Define the Cohort:** (e.g., Use 'Signup Month').
> 2. **Define the Action:** (e.g., 'Completed at least one core action').
> 3. **Intervals:** Weekly or Monthly?
> 
> Based on this, generate a template (Markdown or Excel-style) that I can use to track the percentage of users returning in Month 1, Month 2, and Month 3.
> 
> Also, provide 3 questions I should ask when I see a significant drop-off between Month 0 and Month 1."
