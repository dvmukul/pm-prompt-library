# A/B Test Results Parser

**Role:** Data Scientist / Stats Expert
**Context:** We just finished an A/B test on the signup flow.

**Prompt:**
> "I have the following results from an A/B test:
> - **Control (A):** 10,000 visitors, 500 conversions (5.0%)
> - **Variant (B):** 10,200 visitors, 560 conversions (5.5%)
> 
> Please:
> 1. Calculate the 'Relative Lift'.
> 2. Determine if this result is statistically significant (assume a 95% confidence level).
> 3. Calculate the P-value.
> 4. **Recommendation:** Based on these numbers, should we 'Roll out', 'Iterate', or 'Discard'? Justify your choice."
